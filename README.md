📖 Memorias Errantes
Fragmentos de una misma herida esparcidos por la ciudad como ecos de metal.
Memorias Errantes es un pequeño proyecto de poesía urbana e interactiva.
Nueve textos breves –cada uno una estación de un viaje emocional– se ocultan detrás de códigos. 
Los códigos están grabados en pins físicos que se abandonan en cafés, bancos y rincones de la ciudad.
Quien encuentra un pin, escanea el código QR y desbloquea un fragmento en esta página web.
Cada memoria es un latido, una promesa de belleza en este siglo de ceniza.

Características
  Nueve fragmentos con diseño oscuro y tipografía monoespaciada.
  Buscador manual – introduce el código encontrado en la ciudad para leer cualquier memoria.
  Botón «Compartir» – envía el fragmento por WhatsApp, Telegram, correo, etc. (funciona en móviles y escritorio).
  Botón «Descargar imagen» – genera una imagen cuadrada (800×800) con el texto y la firma del proyecto, lista para Instagram.
  Sin dependencias externas – todo el código es HTML, CSS y JavaScript puro (Canvas nativo para las imágenes).
  Licencia Creative Commons – prohibida su venta, permite compartir sin modificaciones.

Tecnología
  HTML5 + CSS3 (diseño responsive, oscuro y sobrio)
  JavaScript vanilla (sin librerías ni frameworks)
  Canvas API (generación de imágenes sin dependencias)
  Web Share API y Clipboard API (compartir texto)
  GitHub Pages (alojamiento gratuito)
